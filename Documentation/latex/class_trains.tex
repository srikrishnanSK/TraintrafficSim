\doxysubsection{Detailed Description}
The Train class is responsible for updating index and tracksegments for the train to move approproately. Additionally, it is also utilized for setting the trains direction, routes, and also obtain other related status information to update movement accordingly in the \mbox{\hyperlink{class_simulator}{Simulator}} class.
